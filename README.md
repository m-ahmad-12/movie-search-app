MovieHub

A movie search application built with React that allows users to search for movies, view detailed information, and save favorites.

Live Demo
https://movie-search-app-ten-beige.vercel.app/

Features
- Search movies in real-time using the OMDB API
- View detailed movie information (plot, cast, director, rating)
- Add and remove movies to favorites
- Favorites persist across sessions using localStorage
- Trending search suggestions on home page
- Responsive dark cinematic UI

Tech Stack
- React 18
- React Router v6
- Custom Hooks (useFetch, useLocalStorage)
- OMDB API
- CSS Modules
- Deployed on Vercel

Run Locally
git clone https://github.com/m-ahmad-12/movie-search-app.git
cd movie-search-app
npm install
npm run dev
